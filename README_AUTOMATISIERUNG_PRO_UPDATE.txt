Koenen App – Automatisierung & Prüfcenter Professional Update

Verbessert in dieser Version:
- Automatisierungsseite professioneller gestaltet.
- Hausgeld/WEG-Erkennung erweitert: erkennt nun u.a. Hausverwaltung, Hausverwalter, Hausgeld, Wohngeld, WEG, Wirtschaftsplan, Treis und Gies Wohnbau.
- Objekt-Jahresübersicht zeigt Hausgeld/WEG zuverlässiger an, insbesondere Buchungen unter Kategorie/Notiz „Hausverwaltung“.
- Mietcheck verständlicher formuliert: keine kryptische Anzeige mehr wie „/ Monate 5, 6, 7“, sondern Monatsnamen und Soll/Ist-Hinweis.
- Liquiditätsverlauf zeigt jetzt das ausgewählte Jahr im Titel.
- Monatsanzeige im Liquiditätsverlauf zeigt Monatsnamen statt nur Zahlen.
- Navigation oben verbreitert, damit Datenprüfung und Automatisierung optisch auf gleicher Menüebene stehen.
- Bestehende Nebenkosten-, Buchungsimport-, Supabase- und Druckfunktionen bleiben unverändert erhalten.

Build geprüft mit:
npm install
npm run build

Keine neue Supabase-Migration erforderlich.
